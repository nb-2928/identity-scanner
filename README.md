# Identity Scanner Demo

This project is a sample demonstration utility designed for scanning and parsing data from identity documents, such as driver’s licenses and passports. It serves as a proof-of-concept for identity verification workflows.

### Features
* **Document Parsing:** Extracts information from various document formats.
* **Validation Logic:** Demonstrates how to handle and validate scanned ID data.
* **Modular Design:** Built with React, TypeScript, and Vite, featuring a clean architecture suitable for educational purposes.

### Prerequisites
* [Node.js](https://nodejs.org/) (latest LTS recommended)
* Git

### How to Run
1. Clone this repository to your local machine.
2. Install the necessary dependencies:
   ```bash
   npm install
3. Run the development server:
   ```bash
   npm run dev
